Psssing data between UIViewControllers when using UINavigationViewController

这一篇的内容是关于在使用NavigationViewController时，数据是怎么在众多UIViewController之间传递的，包括了从上一个ViewController传给下一个ViewController，也包括了从下一个ViewController传回给上一个ViewController等等。

在前一个ViewController中传递数据给下一个ViewController

使用公开可写的property传递数据

首先是把数据从前一个ViewController传给下一个ViewController，假设上一个ViewController名为PreviousViewController，而下一个ViewController名为NextViewController。

要这样传递数据，首先我们要保证数据在即将接收它的ViewController中是一个公开且可写的property，即将它声明在interface中，且不是readonly。

由于PreviousViewController负责创建NextViewController，那么在创建完NextViewController后，传给NavigationViewController之前，我们需要调用数据property的setter方法，将数据设置好：

// 在负责显示NextViewController的代码中
NextViewController *nextViewController = [[NextViewController alloc] init];
[nextViewController setFoo: someData];
[self.navigationController pushViewController:nextViewController animated:YES];
