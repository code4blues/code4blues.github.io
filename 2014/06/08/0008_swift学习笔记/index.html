Swift学习笔记

Selector跟@selector

Swift的Selector跟ObjC的@selector是一样的，返回一个SEL类型。

不同的是，ObjC的@selector的参数是标识符，而Swift的Selector的参数时字符串。

首先是ObjC的代码：

// Objective-C
UITapGestureRecognizer *doubleTapGesture = [UITapGestureRecognizer alloc]
  initWithTarget:self
          action:@selector(someMethod:)];

// Definition of someMethod:     
- (void)someMethod:(UIGestureRecognizer *)gr {
  // Do something
}

接下来是Swift的代码：

// Swift
let doubleTapGesture = UITapGestureRecognizer(target: self, action:Selector("someMethod:"))

// Definition of someMethod:
func someMethod(gr: UIGestureRecognizer) {
  // Do something
}

其实在Swift中可以把Selector省略，直接用字符串代替：

 // Swift
 let doubTapGesture = UITapGestureRecognizer(target: self, action: "someMethod")

QuartzCore

在Xcode6-beta2中，通过Swift编写CALayer相关代码的时候，比如说

func yRotation(angle: Float) -> CATransform3D {
  ......
}

这个时候会报错：

Use of module 'CATransform3D' as a type

而在Objc里面是没有这样的错误的。

碰到这个错误，我们需要做得就是将QuartzCore模块加载进来。在文件的顶部加载它：

import QuartzCore
