Sketch小技巧

调整圆角矩形的圆角数量

Sketch中画出来的圆角矩形（快捷键U）默认是4个圆角，而有时候我们只需要两个圆角，这时候需要对圆角数量作出调整，采用以下步骤：

  1. 选中圆角矩形
  2. 在工具栏中选中Edit
  3. 在画布中选择需要去掉的圆角
  4. 在检查器中（界面右边栏）调整Corners的数量为0
