代码移动

  • option + 左/右箭头：按单词移动光标

删除

  • option + delete：删除单词
  • option + fn + delete: 反向删除一个单词
  • command + delete：删除光标前所有内容
  • control + k：删除该行光标后的内容

工具面板

  • command + option + 0：展开/关闭工具面板

调试面板

  • command + shift + Y：展开/关闭debug面板

导航面板

  • command + 0：展开/关闭导航面板

Assistant Editor

  • command + option + enter/return：打开Assistant Editor
  • command + enter/return：关闭Assistant Editor
  • option + 鼠标选中文件：在Assistant Editor中打开文件

Tab相关

  • command + t：创建新Tab页
  • command + shift + [：切换到左边Tab页
  • command + shift + ]：切换到右边Tab页
